# Product Listing Card

A responsive product listing component built with CSS Grid.  
- 4 products per row on desktop  
- 2 products per row on mobile  
- Includes SEO basics (title, meta description, semantic tags)

## Features
- Semantic HTML (`h1`, `h2`, `p`, `div`)
- Descriptive class names (`product-card`, `product-name`, `product-price`)
- Responsive design with media queries
- SEO-friendly structure
